LINKA
==============================

* A desktop app for quicking grabbing data relevant to E-Learning stuff.

* Can change student email password but not E-Learning.

* Compiles under .NET 4.0 using, should work better under 4.5 with async/await changes.

* Requires:
  * .NET 4.0 (at least)
  * Oracle.NET library
  * Google Admin SDK for .NET
